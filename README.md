### ЁЯУж Arrays тАФ Searching ЁЯФД In Progress
- тЬЕ **Linear Search** тАФ Traverse array element by element. `O(n)` time, `O(1)` space.

- тЬЕ **Bubble Sort** тАФ Adjacent swaps with early exit optimization. `O(n┬▓)` worst, `O(n)` best.

- тЬЕ **Insertion Sort** тАФ Build sorted portion one element at a time. `O(n┬▓)` worst, `O(n)` best.
- ЁЯФД Merge Sort, Quick Sort coming...

## ЁЯТб How I Approach Each Problem

1. **Understand** тАФ Read the problem carefully. Identify input/output.
2. **Brute Force** тАФ Think of the simplest possible solution first.
3. **Optimize** тАФ Analyze bottlenecks. Apply better data structures or algorithms.
4. **Code** тАФ Write clean, well-commented code.
5. **Test** тАФ Dry-run edge cases (empty array, single element, duplicates, etc.).
6. **Reflect** тАФ Note the time/space complexity and key learnings.
